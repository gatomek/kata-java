TreeMap is a map implementation that keeps its entries sorted according to the natural ordering of its keys or 
better still using a comparator if provided by the user at construction time ^t4w8TyzO

TreeMap, unlike a hash map and linked hash map, does not employ the hashing principle anywhere
since it does not use an array to store its entries ^24Gkpmyf

Notes ^M7XFeEHo

A hash map does not guarantee the order of keys stored and specifically does not guarantee that this order will remain the same over time, 
but a tree map guarantees that the keys will always be sorted according to the specified order. ^lT1N5J0t

Because of this attribute of tree maps, we can perform queries like; find “largest”, find “smallest”,
find all keys less than or greater than a certain value, etc ^hERS9MoN
